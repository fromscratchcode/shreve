# Memphis Playground

Memphis Playground is the browser-based code runner for
[Memphis](https://memphis.fromscratchcode.com/).

This repo powers both:

- the standalone Memphis Playground app
- the `@fromscratchcode/memphis-react` library, which exports `MemphisEmbed`

## Local Development

```bash
npm install
npm run dev
```

## Library Build

```bash
npm run build:lib
```
